    <div class="wikistyle"><h1><a href="http://jquery.com/">jQuery</a> - New Wave Javascript</h1>

<h2>What you need to build your own jQuery</h2>

<ul>
<li><p>Make sure that you have Java installed (if you want to build a minified version of jQuery).
If not, go to this page and download "Java Runtime Environment (JRE) 5.0"<br />
<a href="http://java.sun.com/javase/downloads/index.jsp">http://java.sun.com/javase/downloads/index.jsp</a></p></li>
<li><p>You now have two options for building jQuery, if you have access to common UNIX commands (like <code>make</code>, <code>mkdir</code>, <code>rm</code>, <code>cat</code>, and <code>echo</code>) then simply type <code>make</code> to build all the components.</p></li>
<li><p>The other option is if you have Ant installed (or are on Windows and don't have access to make). You can download Ant from here: <a href="http://ant.apache.org/bindownload.cgi">http://ant.apache.org/bindownload.cgi</a>
If you do have Ant, everytime (in this README) that I say 'make', do 'ant' instead - it works identically (for all intents and purposes).</p></li>
</ul>


<h2>How to build your own jQuery</h2>

<p>In the main directory of the distribution (the one that this file is in), type
the following to make all versions of jQuery:</p>

<p><code>make</code></p>

<p>Here are each of the individual items that are buildable from the Makefile.</p>

<p><code>make init</code></p>

<p>Pull in all the external dependencies (QUnit, Sizzle) for the project.</p>

<p><code>make jquery</code></p>

<p>The standard, uncompressed, jQuery code.<br />
Makes: ./dist/jquery.js</p>

<p><code>make min</code></p>

<p>A compressed version of jQuery (made the Closure Compiler).<br />
Makes: ./dist/jquery.min.js</p>

<p><code>make lint</code></p>

<p>Tests a build of jQuery against JSLint, looking for potential errors or bits of confusing code.</p>

<p><code>make selector</code></p>

<p>Builds the selector library for jQuery from Sizzle.<br />
Makes: ./src/selector.js</p>

<p>Finally, you can remove all the built files using the command:</p>

<p><code>make clean</code></p>

<p>Additionally, if you want to install jQuery to a location that is not this
directory, you can specify the PREFIX directory, for example:</p>

<p><code>make PREFIX=/home/john/test/</code></p>
